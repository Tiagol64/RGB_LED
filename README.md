### Arduino PinOUT

* RGB LED:
  * Anode: GND
  * Red: D9 (220Ω)
  * Green: D10 (220Ω)
  * Blue: D3 (220Ω)
    
* Red Potentiometer:
  * Gnd: GND
  * Output: A3
  * Vcc: VCC

* Green Potentiometer:
  * Gnd: GND
  * Output: A2
  * Vcc: VCC
 
* Blue Potentiometer:
  * Gnd: GND
  * Output: A1
  * Vcc: VCC

* TFT Screen:
  * Gnd: GND
  * Vcc: VCC
  * SCL: D13
  * SDA: D11
  * RST: D8
  * DC: D7
  * CS: D4
  * BL: D5
